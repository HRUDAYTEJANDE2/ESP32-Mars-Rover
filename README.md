# Autonomous Mars Rover Prototype

A simple ESP32 mars rover project I'm building for fun

## Features
- ESP-NOW rover ↔ base station communication
- Wheel encoder odometry
- environmental sensing
- modular rover electronics

## Project Goals
1. Build a functional rover platform
2. Implement basic autonomous navigation
3. Design a custom PCB control board
4. Develop a solar-powered system
